# MEAN Stack Sample


## start for devlopment

```
cd mean-back
npm install
npm start
```

open another terminal.

```
cd mean-front
npm install
npm start
```


## Hint: Generate new application

```
npm install
```

Generate new frontend app.

```
npx ng new --defaults mean-front
```

Generate new backend app.

```
npx express --view=pug --git mean-back
```
